    Invaders++

    As a final project PIC 10C, an advanced course in C++ at UCLA, I built a modern graphical recreation of Space Invaders using Qt. The game features 3 levels with unique enemies, the last being a boss level. If interested, the GitHub repository includes DOxygen HTML and LaTeX files for code details.

    LANPong

    As an effort to learn sockets and networking for game development, I created a version of Pong that can be played over LAN. One player will create a server by specifying a port (try to use ports above 1024 or else it may conflict with other applications). Then anyone else on the network can connect to that player by entering that person's IP address and the same port number. The entire application was created in Java using the AWT and Swing libraries.

    MNIST

    As an introduction to machine learning, I developed a simple Python script to read a page full of handwritten letters and identify each letter. By implementing a support-vector machine, the classification model achieved 99% accuracy using only 20% of the data for training.

    TrackerCV

    In an experiment with computer vision, I developed a script to track and visually highlight an object in a video clip. Additionally, the xy-coordinates of the object in the video frame are plotted to graphically show the travel path of the object over time.

    SleeplessLA

    As a side project, I built a website for my street dance crew Sleepless to help facilitate our event-booking process. The page was built using AngularJS and Node.js and deployed on Heroku.
